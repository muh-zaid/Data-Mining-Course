# Project Overview
This project investigates whether team payroll influences success in Major League Baseball (MLB) using statistical analysis and machine learning techniques. The analysis is implemented in a Google Colab Jupyter Notebook that performs data loading, preprocessing, exploratory analysis, regression modeling, and classification modeling.
The notebook evaluates the relationship between team payroll, Wins Above Replacement (WAR), and team performance outcomes such as winning percentage, playoff appearances, and World Series results.

# Project Structure
The project consists of the following files:
- MLB_Finance_Performance_Analysis.ipynb (main Google Colab notebook containing all Python code)
- mlb_finance_data_1.csv (dataset file used in the project containing MLB team payroll, performance statistics, and outcome variables)

# How to Run the Project
1. Download or clone this repository to your local machine.
2. Open MLB_Finance_Performance_Analysis.ipynb in Google Colab or Jupyter Notebook.
3. Run all cells in the notebook sequentially.
4. When prompted, upload the required dataset file:
   mlb_finance_data_1.csv
5. The notebook will automatically:
  - Load and preprocess the dataset
  - Perform exploratory data analysis
  - Run regression models to analyze payroll vs team performance
  - Train Random Forest models for predicting playoff and championship outcomes
  - Generate visualizations and model evaluation results.
